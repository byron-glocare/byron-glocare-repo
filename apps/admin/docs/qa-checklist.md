2. **대시보드 버킷 오판정** 🔧
   - 기존: 이미 근무중인 고객(c13/c14)이 "강의일정 확정 필요" 버킷에 포함, 대기중 고객(c15)이 "교육원 매칭/예약금" 버킷에 포함
