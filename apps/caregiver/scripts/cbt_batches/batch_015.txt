```
[
  {
    "id": 1740,
    "intent_ko": "...",
    "intent_vi": "...",
    "choice_explanations": {
      "ko": { "1": "...", "2": "★ ..." },
      "vi": { "1": "...", "2": "★ ..." }
    },
    "key_terms": [{ "term_ko": "주보호자", "def_vi": "Người chăm sóc chính" }]
  },
  ... (각 id 마다 한 객체)
]
```

⚠️ 정답에 해당하는 보기의 해설은 반드시 ★ 로 시작 (한국어·베트남어 둘 다)
⚠️ 무의미한 보기는 키 자체 생략 (ko·vi 양쪽 동일하게)
⚠️ 한국어와 베트남어 블록 완전 분리
